Setup Instructions

1. Configure Stripe Webhook

Add this endpoint URL to your Stripe webhook settings:

https://your-worker.workers.dev/webhook/stripe

Subscribe to these events: invoice.payment_failed, invoice.payment_succeeded

2. Set Environment Variables

Run these commands to configure secrets:

wrangler secret put STRIPE_WEBHOOK_SECRET
wrangler secret put TWILIO_ACCOUNT_SID
wrangler secret put TWILIO_AUTH_TOKEN
wrangler secret put TWILIO_WHATSAPP_NUMBER
wrangler secret put OPENAI_API_KEY
wrangler secret put DASHBOARD_API_KEY

3. Test the Integration

Use the Stripe CLI to send a test webhook:

stripe trigger invoice.payment_failed
